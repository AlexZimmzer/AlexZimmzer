Web Application Programming and Hacking

Front-end Web Development Lab

Instructor: Dr. Phu Phung

Student: Alex Zimmer

In this course, we explore basic web development applications including HTML, JS, CSS, PHP, and MySQL.
We also look at vulnerabilities and attacks using hackathon assignments.
The course repository is linked here.

Lab 1

In this lab, we were introduced to HTTP (Hypertext Transfer Protocol), Wireshark, CGI (Common Gateway Interface) applications, and GET/POST requests. This lab was focused on introducing basic web application.

Lab 2

In this lab, we worked with basic HTML, JavaScript, Ajax, CSS, jQuery, and Web API integration to create an information page. It included a digital clock, GET and POST request forms, and integration of JokeAPI.

Hackathon 1

In this hackathon, we looked at XSS (Cross-site Scripting Attacks) and defenses. To do this, we worked through various exercises on the course Apache page and used various methods including using GET/POST requests to display an alert on the page. We also provided the webpage we created in Lab 2 with defenses to prevent these attacks.
